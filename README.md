# Portscanner
This portscanner is created by **0x47root**.

## Usage:
```bash
python3 portscanner_niveau1.py
```

## Possible scan types
- TCP-Connect Scan (-sT)
- UDP Scan (-sU)
- SYN Scan (-sS)
- XMAS Scan (-sX)

## Additional functionalities
- Scan range of ports
- Save scan results to XML or JSON
- Automatically save results to SQLite database
